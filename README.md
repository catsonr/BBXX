# BBXX GAME ENGINE

**BBXX** is a cross-platform game engine, written in c++

see **https://github.com/catsonr/beatboxx** for version 1.0

## install

`git clone --recurse-submodules https://github.com/catsonr/bbxx/`

(this may take a while due to submodules)

## compile

`cd bbxx`

`cmake -B build`

`cmake --build build`

## run

`./build/Debug/bbxx`
